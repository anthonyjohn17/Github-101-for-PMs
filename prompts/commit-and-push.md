# Prompt: Commit and Push

Copy and paste into Cursor or Claude Code after you've made changes.

---

**After building or updating code:**
```
Commit all changes with message: "[verb] [what you did]"
Then push to GitHub.
```

**Good commit messages:**
- "Add email drafting agent with OpenAI integration"
- "Fix API key validation error"
- "Update README with setup instructions and demo GIF"
- "Add sentiment analysis to feedback clustering"

**Bad commit messages:**
- "Updates"
- "Fixed stuff"
- "WIP"
- "Changes"

---

**Other common operations:**

Undo last commit (keep the files):
```
Undo the last commit but keep my changes
```

Save work without committing:
```
Stash my current changes
```

Restore stashed work:
```
Apply my stashed changes
```

Create a branch to experiment:
```
Create a new branch called "experiment-[feature-name]"
```
